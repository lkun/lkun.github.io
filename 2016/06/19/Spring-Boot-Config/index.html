Spring Boot 配置文件

Spring Boot 使用一个全局的配置文件 applications.properties,放置在 src/main/resources 目录或者类路径下面的 /config下。
Spring Boot 不仅支持常规的 properties配置文件，还支持yaml语言的配置文件。yaml是以数据为中心的语言，在配置数据的时候具有面向对象的特征。
Spring Boot 的全局配置文件的作用是对一些默认配置的配置值进行修改。

简单实例

将Tomcat的默认端口号8080修改为9090，并将默认的访问路径”/“修改为”/helloworld”。
可以在application.properties 中添加：

server.port=9090
server.context-path=/helloworld

或者在 application.yml中添加：

server:
    port: 9090
    contextPath: /helloworld

定制Banner

  • 修改Banner
    在Spring Boot 启动的时候会有一个默认的启动图案Spring
  • 在src/main/resources 下新建一个banner.txt
  • 通过 http://patorjk.com/software/taag 网站生成字符，如敲入的为 “lvkun”,将网站生成的字符复制到banner.txt中。
  • 启动的时候你将会看到图案是lvkun

关闭banner

  • main 里的内容修改为：

    SpringApplication app = new SpringApplication(Demo.class);
app.setShowBanner(false);
app.run(args);

  • 或使用fluent API 修改为：

    new SpringApplicationBuilder(Demo.class).showBanner(false).run(args);
